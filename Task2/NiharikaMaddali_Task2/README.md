## 📌 Project Overview

This project performs **Customer Segmentation Analysis** on an e-commerce marketing dataset using **K-Means Clustering**. By analyzing customer behavior, demographics, and purchase patterns, we group customers into distinct segments to help businesses design targeted marketing strategies and improve customer satisfaction.

> **Internship Program:** Data Analytics Internship — Oasis Infobyte
> **Project Level:** Level 1 — Project 2

---

## 🎯 Objectives

- Analyze customer demographics and purchase behavior
- Identify meaningful customer segments using unsupervised machine learning
- Visualize clusters to extract actionable business insights
- Provide data-driven recommendations for targeted marketing

## 📂 Dataset

| Detail | Info |
|--------|------|
| File Name | `ifood_df.csv` |
| Source | [Kaggle — Marketing Analytics: Customer Segmentation](https://www.kaggle.com/) |
| Records | ~2,200+ customers |
| Features | 35+ variables |

### Key Features in Dataset

| Feature | Description |
|---------|-------------|
| `Education` | Customer's education level |
| `Marital` | Customer's marital status |
| `AcceptedCmp1–5` | Whether customer accepted campaigns 1 to 5 |
| `Response` | Target — accepted the last campaign |
| `Complain` | Whether customer complained in last 2 years |
| `Kidhome` | Number of small children at home |
| `Teenhome` | Number of teenagers at home |
| `DtCustomer` | Date of customer enrollment |

---

## 📊 Key Steps Explained

### 1. Data Cleaning
- Handled missing values and null entries
- Removed duplicate records
- Standardized data types and formats

### 2. Exploratory Data Analysis
- Analyzed distribution of customer demographics
- Studied purchase frequency and campaign responses
- Generated correlation heatmaps

### 3. K-Means Clustering
- Used the **Elbow Method** to determine optimal number of clusters (K)
- Applied K-Means algorithm to segment customers
- Labeled each customer with their cluster group

### 4. Visualization
- Scatter plots to display customer segments
- Bar charts comparing cluster characteristics
- Heatmaps for feature correlations

---

## 💡 Business Insights

- Customers with higher education tend to respond better to premium campaigns
- Customers with children show different purchase patterns compared to those without
- Certain segments show consistently high campaign acceptance rates
- Complaint history correlates with lower future campaign engagement
